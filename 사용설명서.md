# 📦 클린 바코드 재고관리 프로그램 사용 설명서

틸인스 / 보노 제품의 바코드 입출고와 재고를 관리하는 웹 기반 프로그램입니다.

---

## 1. 빠른 시작 (3분 안에 사용 시작)

1. `재고관리.html` 파일을 **더블클릭** 하거나, 크롬(Chrome) 브라우저로 엽니다.
2. 처음 열면 틸인스 6개 제품이 샘플로 들어와 있습니다.
3. **상품관리 탭**에서 진짜 바코드와 초기 재고를 입력합니다.
4. **스캔/입출고 탭**에서 바코드를 스캔하고 입출고를 처리합니다.

> 💡 **중요**: 데이터는 브라우저에 저장됩니다. 다른 PC에서 보려면 구글시트 연동을 설정하세요.

---

## 2. 화면 구성

| 탭 | 용도 |
|---|---|
| 🔍 **스캔/입출고** | 바코드 스캔 → 입출고/위치/수량 입력 → 처리 |
| 📊 **재고현황** | 사무실/창고별 실시간 재고 확인, CSV 내보내기 |
| 📋 **거래내역** | 전체 입출고 이력 조회, 검색, CSV 내보내기 |
| 🏷️ **상품관리** | 상품 추가/수정/삭제, CSV 일괄 등록 |
| ⚙️ **설정** | 구글시트 연동, 백업/복원, 초기화 |

---

## 3. 입출고 처리 방법

1. 🔍 **스캔/입출고** 탭으로 이동
2. 상품을 선택하는 4가지 방법:
   - **📷 카메라 시작** 버튼으로 바코드 촬영
   - **USB 바코드 스캐너**: "🔎 상품 검색" 칸을 클릭한 상태로 스캔
   - **상품명/검색어 검색**: "🔎 상품 검색" 칸에 직접 입력 (예: `1구`, `티원 화이트`, `4구 보노`)
     - 결과가 1개면 자동 선택, 여러 개면 목록에서 클릭
     - 공백으로 여러 단어 입력하면 모두 포함된 상품 찾기 (예: `스위치 4구`)
   - **빠른 선택**: 화면 하단에서 최근/자주 쓰는 상품 클릭
3. 상품이 자동 선택되면:
   - **📥 입고 / 📤 출고** 선택
   - **🏢 사무실 / 🏭 창고** 위치 선택
   - **수량 입력** (+/- 버튼 또는 직접 입력)
   - **메모** 입력 (선택사항: 거래처명 등)
4. **✅ 처리하기** 클릭 → 재고 자동 업데이트, 거래 내역 저장

> 출고 시 재고보다 많은 수량을 입력하면 경고가 뜹니다 (음수 재고도 허용 가능).

### 💡 검색어 활용 팁
상품관리 탭에서 각 상품에 **검색어**(쉼표로 구분)를 등록해두면 빠르게 찾을 수 있습니다.
- 예) `틸인스 스위치 1구 [Ti1 화이트]` → 검색어: `스위치, 1구, ti1, 화이트, 틸인스`
- 이렇게 등록하면 `1구`, `티원`, `화이트` 등 다양한 키워드로 찾을 수 있습니다.
- 같은 검색어가 여러 상품에 있으면(예: `스위치`) 결과 목록에서 골라서 선택합니다.

## 4. 구글시트 연동 설정 (한 번만 하면 됩니다)

### 4-1. 구글시트 만들기
1. [Google Sheets](https://sheets.google.com) 접속
2. 새 시트 만들기 → 이름을 **"클린 재고관리"** 등으로 변경

### 4-2. Apps Script 코드 붙여넣기
1. 시트 상단 메뉴: **확장 프로그램 → Apps Script**
2. 기본으로 들어있는 `function myFunction() { }` 코드를 **모두 삭제**
3. 제공된 `구글시트_AppsScript.gs` 파일의 내용을 **전체 복사**해서 붙여넣기
4. 💾 **저장** (Ctrl+S 또는 Cmd+S), 프로젝트 이름은 "재고관리" 등 자유롭게

### 4-3. 웹 앱 배포
1. 오른쪽 상단 **배포 → 새 배포** 클릭
2. ⚙️ 톱니바퀴 → **웹 앱** 선택
3. 설정:
   - **설명**: 재고관리 (자유)
   - **다음 사용자로 실행**: **나(본인 이메일)**
   - **액세스 권한**: **모든 사용자**
4. **배포** 버튼 클릭
5. 권한 승인 (Google 계정으로 로그인 → 고급 → 안전하지 않음으로 이동 → 허용)
6. **웹 앱 URL** 복사 (`https://script.google.com/macros/s/...../exec`)

### 4-4. HTML에 URL 입력
1. `재고관리.html`을 열고 ⚙️ **설정** 탭으로 이동
2. **Web App URL** 칸에 위에서 복사한 URL 붙여넣기
3. **URL 저장** 클릭
4. **연결 테스트** 클릭 → "✅ 연결 성공" 메시지 확인
5. **전체 업로드** 클릭 → 현재 데이터를 시트에 보냅니다.
6. **자동 동기화** 체크박스를 켜면 거래마다 시트에 자동 반영됩니다.

> 💡 첫 배포 후 Apps Script 코드를 수정했다면, **배포 → 배포 관리 → 연필 아이콘 → 버전: 새 버전 → 배포** 를 다시 해야 반영됩니다.

## 5. 자주 묻는 질문

**Q. 카메라가 안 켜져요.**
- 크롬에서 열었는지 확인 (사파리는 일부 환경에서 제약 있음)
- URL이 `file://`인 경우 일부 브라우저는 카메라를 차단합니다. → **USB 바코드 스캐너**를 쓰는 것이 안정적입니다.
- USB 스캐너는 그냥 꽂으면 키보드처럼 동작하므로 "바코드 직접 입력" 칸을 클릭한 상태에서 스캔만 하면 됩니다.

**Q. 데이터가 사라졌어요.**
- 브라우저의 시크릿 모드/방문 기록 삭제 시 데이터가 날아갈 수 있습니다.
- ⚙️ **설정 → 전체 백업** 으로 주기적으로 JSON 파일을 받아두세요.
- 구글시트 연동을 설정해두면 안전합니다.

**Q. 두 대 이상의 컴퓨터에서 함께 쓰고 싶어요.**
- 같은 구글시트 URL을 각 PC에 입력하면 됩니다.
- 다만 동시에 여러 곳에서 거래를 하면 충돌이 날 수 있으니, 한 번에 한 명만 쓰는 것을 권장합니다.
- 작업 시작 전 ⚙️ 설정 → **전체 다운로드** 를 누르면 시트의 최신 상태를 받아옵니다.

**Q. 상품을 한꺼번에 등록하고 싶어요.**
1. 🏷️ **상품관리 → CSV 내보내기** 로 빈 양식을 받아옵니다.
2. 엑셀에서 `상품명, 바코드, 사무실, 창고` 컬럼을 채웁니다.
3. **CSV 가져오기** 로 업로드합니다. (중복 바코드는 자동 스킵)

**Q. 잘못 처리했어요.**
- 📋 **거래내역** 탭에서 확인 가능합니다.
- 잘못된 거래는 반대 거래(입고↔출고)로 상쇄해주세요.
- 상품관리 탭에서 직접 재고 수치를 수정해도 됩니다.

```
├── 구글시트_AppsScript.gs ← 구글시트에 붙여넣을 코드

```

## 7. 기술 사양

- **저장**: 브라우저 localStorage + 구글시트(Apps Script)
- **바코드 인식**: html5-qrcode (EAN-13, Code-128, QR 등 대부분 지원)
- **외부 의존성**: 인터넷 연결 (라이브러리 로드 + 시트 동기화 시에만)
- **호환 브라우저**: Chrome, Edge, Safari (최신 버전)
- **오프라인**: 인터넷 없이도 로컬 사용 가능 (시트 동기화만 불가)
